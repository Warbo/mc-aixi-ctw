Release Notes:
--------------

14/7/2010 - Initial Release of mc-aixi(fac-ctw)

Introduction:
-------------

A prototype implementation of an approximate AIXI agent based on:
"A Monte Carlo AIXI Approximation"
Joel Veness, Kee Siong Ng, Marcus Hutter, William Uther, David Silver

See http://jveness.info/publications/default.html for more detail.


Compilation Instructions:
-------------------------
 - Install Boost for C++: www.boost.org
 - Install John Torjo's Boost Logging Library: http://torjo.com/log2/
 - Compile with MSVC 8+/GCC 4+. Untested with other C++ compilers.


Getting started:
-----------------

./pipsqueak --help
